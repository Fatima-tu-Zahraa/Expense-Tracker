# Expense Tracker

A personal expense tracker web application built with React.js and Tailwind CSS.

## Live Demo
[View Live](https://expense-tracker-seven-tawny-84.vercel.app)

## Features
- Add and delete expenses
- Category filter (Food, Transport, Shopping, Bills, Other)
- Set monthly budget with limit warning
- Remaining balance tracker
- Data saved with Local Storage (persists on refresh)

## Tech Stack
- React.js
- Tailwind CSS
- Vite
- Local Storage

## Getting Started
```bash
npm install
npm run dev
```
